Demo of the web version of [ChimpGB](https://github.com/tomas7770/chimp-gb).

**Note:** This is in an experimental state.

Includes the following games:

- [Tobu Tobu Girl Deluxe](https://github.com/SimonLarsen/tobutobugirl-dx)
- [GBSnake](https://github.com/Yvar-deGoffau/GBSnake/tree/master)
- [2048-gb](https://github.com/Sanqui/2048-gb)

Alternatively, you can provide a Game Boy ROM file to load in the emulator.
